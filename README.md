# AI Bookmarks

This repo stores list of AI tools, products, and frameworks I have explored or looking to explore.

## Locally Run LLMs

- Ollama https://ollama.com/
- LM Studio https://lmstudio.ai/

Both Ollama and LM Studio offers APIs - compatible with Langchain, Agno, etc.

## Multiple LLMs in One

- Chatllm: https://apps.abacus.ai/chatllm/
- Typingmind: https://www.typingmind.com/
- T3Chat: https://t3.chat/

## Python Agent frameworks

- Agno: https://docs.agno.com/introduction - Personal favorite
- Pydantic AI: https://ai.pydantic.dev/
- Google Agent Development Kit: https://google.github.io/adk-docs/

## Vibe Coding

## MCP & Agent Protocols

- Official server list: https://github.com/modelcontextprotocol/servers
- Awesome MCP Servers: https://github.com/punkpeye/awesome-mcp-servers
- Inspector Grizzly: https://github.com/alpic-ai/grizzly: Visual tool for testing MCP servers
- FastMCP: https://github.com/jlowin/fastmcp : MCP Server builder
- Google A2A (agent to agent): https://github.com/google-a2a/A2A : An open protocol enabling communication and
  interoperability between opaque agentic applications.
