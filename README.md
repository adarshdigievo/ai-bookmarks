# AI Bookmarks

This repo stores list of AI tools, products, and frameworks I have explored or looking to explore.

## Locally Run LLMs

- Ollama: https://ollama.com/
- LM Studio: https://lmstudio.ai/

Both Ollama and LM Studio offer APIs — compatible with LangChain, Agno, etc.

## Multiple LLMs in One

- ChatLLM: https://apps.abacus.ai/chatllm/
- TypingMind: https://www.typingmind.com/
- T3Chat: https://t3.chat/
- OpenRouter: https://openrouter.ai/ — Unified API gateway to 200+ LLMs with a single key

## Python Agent Frameworks

- Agno: https://docs.agno.com/introduction — Personal favorite
- Pydantic AI: https://ai.pydantic.dev/
- Google Agent Development Kit (ADK): https://google.github.io/adk-docs/
- LangGraph: https://github.com/langchain-ai/langgraph — Stateful agent orchestration by LangChain
- CrewAI: https://www.crewai.com/ — Multi-agent role-based framework
- smolagents: https://github.com/huggingface/smolagents — Barebones agent library by Hugging Face

## Agent Sandboxing & Runtimes

- Pydantic Monty: https://pydantic.dev/articles/pydantic-monty — Minimal secure Python sandbox for AI agents (
  Rust-based)
- E2B: https://github.com/e2b-dev/E2B — Open source cloud runtime for AI apps & agents
- Container Use: https://github.com/dagger/container-use — Dev environments for coding agents

## Vibe Coding IDEs

- Windsurf: https://windsurf.com/editor — Currently my primary IDE
- Cursor: https://www.cursor.com/
- Trae: https://www.trae.ai/ — Free AI-native IDE by ByteDance
- Kiro: https://kiro.dev/ — Spec-driven agentic IDE by AWS

## Coding Extensions & Skill Packs

- Cline: https://github.com/cline/cline — Open-source coding agent (VS Code extension)
- Superpowers: https://github.com/obra/superpowers — Agentic skills framework for AI coding agents
- gstack: https://github.com/garrytan/gstack — Garry Tan's (YC) Claude Code setup: 15+ agent skills for shipping fast

## CLI Coding Agents

Terminal-native agents that work directly in your shell.

- Claude Code: https://claude.com/product/claude-code — Anthropic's CLI coding agent
- OpenAI Codex CLI: https://github.com/openai/codex — Open-source terminal coding agent
- Gemini CLI: https://developers.google.com/gemini-code-assist/docs/gemini-cli — Google's terminal agent (free tier)
- Aider: https://aider.chat/ — AI pair programming in your terminal, supports many LLMs
- Pi: https://github.com/badlogic/pi-mono — Minimal, opinionated terminal coding agent
- OpenCode: https://opencode.ai/ — Open source AI coding agent for the terminal

## Async Coding Agents

Works by connecting to a GitHub repo and taking up tasks.

- Codex by OpenAI: https://chatgpt.com/codex
- Google Jules: https://jules.google/

## Personal AI Assistants

- OpenClaw: https://openclaw.ai/ — Open-source personal AI assistant & autonomous agent. Connects to Telegram, Discord,
  WhatsApp. Runs on your own server.
